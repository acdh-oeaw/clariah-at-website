Innerhalb des Projekts wurden drei Use Cases entwickelt, die die Anwendung von maschinellem Lernen auf Bibliotheksdatensätze illustrieren. Die Verarbeitung der Daten erfolgt dabei mithilfe des International Image Interoparability Framefowrks (IIIF), eines weit verbreiteten Standards zur Bereitstellung von digitalen Objekten, sodass die entwickelten Anwendungen auch für Ressourcen außerhalb der ÖNB genutzt werden können. Es wurden die folgenden Use Cases ausgearbeitet:

*   Kolorierung von historischen Postkarten, inkl. Anleitung zur Erstellung eines eigenen Kolorierung-Modells
*   Extraktion von Abbildungen
*   Klassifizierung von Abbildungen mittels [Resnet18](https://huggingface.co/Axon/resnet18-v1)
